# Invoice Generator Desktop App (C#)

> **Language:** Dutch (Nederlands)  
> **Note:** This application was developed for a client and is intended for professional business use.

## üìÑ Overview

This project is a **standalone Windows desktop application** developed in **C# (.NET)**.  
The user interface and invoice output are **fully in Dutch**, making it suitable for businesses operating in the Netherlands.

The application allows users to generate professional invoices by entering company and customer details, adding invoice items, automatically calculating VAT (BTW), and exporting invoices as PDF files.

The software works **fully offline** and stores invoices locally on the system.

---

## üéØ Purpose

The purpose of this application is to:
- Generate professional invoices quickly
- Automatically calculate VAT (BTW) and totals
- Maintain consistent invoice numbering
- Reduce manual calculation errors
- Store invoices locally for record keeping

Ideal for **Dutch small businesses, freelancers, and e-commerce sellers**.
